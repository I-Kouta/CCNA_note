# IOS入門

### `インターフェイスの設定`

- インターフェイス設定モード  
`(config)#interface [type] [port]`

- IPアドレスの設定  
`(config-if)#ip address [ip-address] [subnet-mask]`

- インターフェイスの有効化  
`(config-if)#no shutdown`
